### **JavaScript Roadmap (2 Months)**

**Goal:** Master **JavaScript** to build **interactive, dynamic web applications** and use it effectively in **PHP & WordPress development**.

---

## **Phase 1: JavaScript Basics (2 Weeks)**

### **Topics to Cover**

✔ JavaScript Syntax & Variables (`var`, `let`, `const`)  
✔ Data Types (`string`, `number`, `boolean`, `array`, `object`)  
✔ Operators (`+`, `-`, `*`, `/`, `===`, `!=`, `&&`, `||`)  
✔ Conditional Statements (`if`, `else if`, `switch`)  
✔ Loops (`for`, `while`, `do-while`)  
✔ Functions (`function()`, `return`, `arrow functions`)

### **Hands-on Tasks**

✅ Create a **simple calculator** using JavaScript  
✅ Write a **program to check even/odd numbers**  
✅ Build a **basic to-do list (without storage)**

### **Common Mistakes to Avoid**

❌ Using **`var` instead of `let` or `const`** (causes scope issues)  
❌ Not **initializing variables** before use  
❌ Forgetting **`return` in functions** when needed

---

## **Phase 2: DOM Manipulation (2 Weeks)**

### **Topics to Cover**

✔ Selecting Elements (`getElementById`, `querySelector`)  
✔ Changing Content & Styles (`textContent`, `innerHTML`, `style`)  
✔ Handling Events (`onclick`, `addEventListener`)  
✔ Form Validation (`onSubmit`, `preventDefault()`)

### **Hands-on Tasks**

✅ Build a **click counter** (Button increases count)  
✅ Create a **simple form validation system**  
✅ Make a **light/dark mode toggle**

### **Common Mistakes to Avoid**

❌ Using `innerHTML` unsafely (can lead to **security issues**)  
❌ Forgetting to use `preventDefault()` in form submissions  
❌ Overusing `querySelectorAll` when a single element is needed

---

## **Phase 3: JavaScript Arrays & Objects (2 Weeks)**

### **Topics to Cover**

✔ Arrays (`push`, `pop`, `shift`, `unshift`, `map`, `filter`, `reduce`)  
✔ Objects (`{ key: value }`, accessing properties, methods)  
✔ JSON (`JSON.stringify()`, `JSON.parse()`)  
✔ Local Storage (`localStorage.setItem`, `getItem`, `removeItem`)

### **Hands-on Tasks**

✅ Store & retrieve **user data using LocalStorage**  
✅ Filter an **array of numbers** (e.g., find even numbers)  
✅ Build a **product catalog using objects**

### **Common Mistakes to Avoid**

❌ Using **`forEach` when `map` or `filter` is needed**  
❌ Not checking for **null/undefined** before accessing object properties  
❌ Forgetting to **convert JSON data (`JSON.parse()`)** when reading from storage

---

## **Phase 4: Asynchronous JavaScript (2 Weeks)**

### **Topics to Cover**

✔ Callbacks (`setTimeout`, `setInterval`)  
✔ Promises (`.then()`, `.catch()`)  
✔ Async/Await (`async function`, `await fetch()`)  
✔ Fetch API (`fetch()`, `response.json()`)

### **Hands-on Tasks**

✅ Fetch **random user data from an API**  
✅ Display **live time updates on a webpage**  
✅ Create a **weather app using Fetch API**

### **Common Mistakes to Avoid**

❌ Not **handling errors in fetch requests** (`.catch()`, `try...catch`)  
❌ Using **callbacks instead of Promises** (causes callback hell)  
❌ Forgetting to use **`await` inside `async` functions**

---

## **Phase 5: JavaScript in WordPress & Advanced Topics (2 Weeks)**

### **Topics to Cover**

✔ Using JavaScript in WordPress (`wp_enqueue_script`)  
✔ jQuery Basics (`$(document).ready()`, `.click()`, `.hide()`)  
✔ ES6+ Features (`destructuring`, `spread/rest operators`)  
✔ Modules (`import/export`)

### **Hands-on Tasks**

✅ Add **JavaScript validation to a WordPress form**  
✅ Create a **jQuery-powered image slider**  
✅ Use **ES6 modules in a JavaScript project**

### **Common Mistakes to Avoid**

❌ Using **jQuery unnecessarily** (Modern JavaScript can do most things)  
❌ Forgetting to **enqueue scripts properly in WordPress**  
❌ Not using **modern ES6+ features** for cleaner code
