### **PHP Roadmap (3 Months) – Focused on rtCamp Job Preparation**

**Goal:** Master **PHP** for WordPress development, open-source contributions, and the rtCamp software engineer trainee job.

---

## **Phase 1: PHP Basics (3 Weeks)**

### **Topics to Cover**

✔ PHP Syntax & Basics (`echo`, `variables`, `data types`)  
✔ Operators (`+`, `-`, `*`, `/`, `==`, `!=`, `&&`, `||`)  
✔ Conditional Statements (`if`, `else`, `switch`)  
✔ Loops (`for`, `while`, `foreach`)  
✔ Functions (`function`, `return`, `parameters`, `default values`)

### **Hands-on Tasks**

✅ Print **"Hello, World!"** in PHP  
✅ Build a **simple calculator** (Addition, Subtraction, Multiplication, Division)  
✅ Create a **function that returns the sum of an array**

### **Common Mistakes to Avoid**

❌ Using **`echo` instead of `return`** inside functions  
❌ Forgetting **semicolon (`;`) at the end of a statement**  
❌ Not handling **empty arrays** before calculating sums

---

## **Phase 2: Working with Forms, Strings & Arrays (2 Weeks)**

### **Topics to Cover**

✔ Superglobals (`$_GET`, `$_POST`, `$_REQUEST`)  
✔ Form Handling (`isset()`, `empty()`, `htmlspecialchars()`)  
✔ String Functions (`strlen`, `str_replace`, `substr`)  
✔ Array Functions (`array_push`, `array_merge`, `array_filter`)

### **Hands-on Tasks**

✅ Build a **contact form** (Name, Email, Message) with validation  
✅ Create a **search feature** (Filter an array based on user input)  
✅ Develop a **string manipulation tool** (Count characters, replace words)

### **Common Mistakes to Avoid**

❌ Not **validating form inputs** (Leads to security risks)  
❌ Using **`$_POST` directly without checking `isset()`**  
❌ Forgetting to **sanitize input (`htmlspecialchars()`)**

---

## **Phase 3: File Handling & Sessions (2 Weeks)**

### **Topics to Cover**

✔ File Handling (`fopen`, `fwrite`, `fread`, `fclose`)  
✔ Sessions & Cookies (`session_start()`, `setcookie()`, `$_SESSION`)  
✔ Uploading Files (`move_uploaded_file()`, `$_FILES`)

### **Hands-on Tasks**

✅ Build a **guestbook system** (Users can submit messages saved in a file)  
✅ Create a **session-based login system**  
✅ Implement **image upload functionality**

### **Common Mistakes to Avoid**

❌ Not checking **file size/type** before uploading  
❌ Forgetting **`session_start()` at the beginning of session-based pages**  
❌ Not closing files properly after writing/reading

---

## **Phase 4: Object-Oriented PHP (2 Weeks)**

### **Topics to Cover**

✔ Classes & Objects (`class`, `public`, `private`, `constructor`)  
✔ Inheritance & Polymorphism (`extends`, `parent::method()`)  
✔ Magic Methods (`__construct`, `__destruct`)

### **Hands-on Tasks**

✅ Build a **User class** (With login & registration methods)  
✅ Create a **Product class** (With methods to calculate discounts)  
✅ Develop a **blog post system** (Using OOP structure)

### **Common Mistakes to Avoid**

❌ Not using **`$this->property` inside class methods**  
❌ Forgetting to use **constructors for initializing values**  
❌ Overcomplicating **simple functions with unnecessary OOP**

---

## **Phase 5: MySQL & Database Handling (3 Weeks)**

### **Topics to Cover**

✔ Connecting to MySQL (`mysqli_connect`, `PDO`)  
✔ CRUD Operations (`SELECT`, `INSERT`, `UPDATE`, `DELETE`)  
✔ Preventing SQL Injection (`prepared statements`)

### **Hands-on Tasks**

✅ Build a **user authentication system (Login/Register)**  
✅ Create a **task management system** (Add, update, delete tasks)  
✅ Develop a **basic blog with a database**

### **Common Mistakes to Avoid**

❌ Not using **prepared statements** (Leads to SQL injection attacks)  
❌ Forgetting **`mysqli_close()` after database operations**  
❌ Using **old MySQL functions (`mysql_query()`) instead of `mysqli/PDO`**

---

## **Phase 6: WordPress Development with PHP (3 Weeks)**

### **Topics to Cover**

✔ WordPress Theme Development (`functions.php`, `template hierarchy`)  
✔ WordPress Plugins (`wp_enqueue_script`, `shortcodes`, `hooks`)  
✔ Custom Post Types & Meta Fields

### **Hands-on Tasks**

✅ Build a **custom WordPress theme**  
✅ Create a **WordPress plugin** that adds a custom widget  
✅ Develop a **custom post type (Portfolio, Testimonials, etc.)**

### **Common Mistakes to Avoid**

❌ Editing **WordPress core files** instead of using **hooks & filters**  
❌ Not using **wp_enqueue_script()** for loading scripts properly  
❌ Forgetting to **sanitize user inputs in WordPress plugins**
